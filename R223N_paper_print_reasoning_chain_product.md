## 一、迁移判断

本轮不是生成正式教案，而是验证同一课堂事件展开标准能否承接材料 / 技法 / 印痕探究类型。核心链条为：

```text
学情与课标实践要求
→ 大观念：印痕是版画的独特语言
→ 基本问题：纸材和印法碰撞带来怎样的惊喜
→ 表现性任务：版画车间坊
→ 纸材观察 / 肌理制造 / 试印记录 / 印法比较 / 作品展评
→ 作品 + 试印证据 + 展示说明
```

## 二、源证据与推理约束

| 来源节点 | 源样本证据 | 对课堂事件的约束 |
| --- | --- | --- |
| 源样本主题 | 印痕是版画的独特语言。 | 确定本课不是普通纸手工，而是版画印痕语言学习。 |
| 学情 | 三年级学生对肌理有初步认识，在寻找不同质感纸材和拓印材料肌理上有经验，但专业油墨制作、多版多色技能仍需练习。 | 要求课堂事件保留触摸、试印、比较、记录等低门槛动作。 |
| 基本问题 | 不同纸材和不同印法的碰撞会给我们带来怎样的惊喜呢。 | 驱动纸材与印法碰撞后的印痕比较。 |
| 表现性任务 | 用版画作品装饰“版画车间坊”，并形成纸材印痕科普读本、印痕手册或作品展示。 | 要求作品与试印证据共同进入展示，而不是只看成品。 |
| 评价方案 | 能尝试各种纸材制作版画，了解纸材纹理、印痕及特性；能掌握干印、水印、油印等技法；能记录、分享并清理油墨。 | 推动学习单、试印纸、作品照片、展示说明成为证据链。 |

## 三、课堂事件链

| event_id | 课堂事件 | 教学责任 | 最低证据 |
| --- | --- | --- | --- |
| paper_print_event_01_task_entry | 任务入场：走进版画车间坊 | 把课堂从普通纸手工拉回版画语言，明确本课要寻找纸材、印法和印痕之间的关系。 | 能说出一个印痕观察点。 |
| paper_print_event_02_material_texture_observation | 摸一摸：纸材肌理观察 | 让学生用手摸、眼看、侧光观察等方式，把纸材差异转化为可以预测印痕的依据。 | 每组至少一条带理由的纸材预测。 |
| paper_print_event_03_texture_making | 变一变：制造纸的新肌理 | 让学生理解纸材不是被动材料，可以通过改造制造新的可印肌理。 | 每组保留一个改造小样。 |
| paper_print_event_04_first_print_trial | 试一试：第一次转印 | 把纸材预测转化为可观察的印痕证据，让学生建立试印、比较、记录的基本习惯。 | 每组至少一张有圈画和一句调整计划的试印记录。 |
| paper_print_event_05_print_method_comparison | 比一比：干印、湿印与油印 | 让学生在比较中理解印法不是名称记忆，而是会改变印痕清晰度、灰层和画面效果的选择。 | 每组至少一组同版不同印法对照。 |
| paper_print_event_06_artwork_creation_with_print_marks | 做一做：用印痕完成作品 | 引导学生把前面的纸材、肌理、印法选择转化为有画面意识的作品，而不是试印纸堆叠。 | 作品中至少有一处可说明来源的印痕。 |
| paper_print_event_07_gallery_reflection | 说一说：印痕展评与车间收束 | 把展示从“谁的作品漂亮”转为纸材、印法、印痕效果和记录证据的综合说明。 | 每组至少一份作品与试印证据配套归档。 |

## 四、不过拟合说明

本轮没有迁移前一标准样本中的具体内容、仪式或角色设定。组件和大屏只从纸材、印法、印痕、试印记录与作品展评这些源样本要素中触发。
